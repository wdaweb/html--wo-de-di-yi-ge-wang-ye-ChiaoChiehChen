HTML-負責架構、內容


超文本標記語⾔（英語：HyperText Markup Language，簡稱：HTML）是一種基於建立網頁的標準標記語言。 HTML 是一種基礎技術，常與 CSS、JavaScript 一起被眾多網站用於設計賞心悅目的網頁、網頁應用程式以及行 動應用程式的使用者介面。網頁瀏覽器可以讀取 HTML 檔案，並將其彩現成視覺化網頁。HTML 描述了一個網站 的結構語意隨著線索的呈現，使之成為⼀種標記語言而非程式語言。


基礎結構：

      
        <html>
            <head>

            </head>
            <body>

            </body>
        </html>

標籤特性：

  1. 標籤不分大小寫，但請用小寫。
  2. "<>"開頭，"</>"結尾。
  3. 標籤具有屬性與值的特性。
  4. 標籤標記內容，產生語義，組合成了結構。

CSS-負責外觀、樣式


層疊樣式表(英語：Cascading style sheets，簡稱CSS)，又稱串樣式列表、級聯樣式表、串接樣式表、階層式樣式表，一種用來為結構化文件(如HTML文件或XML應用)添加樣式(字型、間距和顏色等)的電腦語言，由W3C定義和維護。目前最新版本是CSS2.1，為W3C得推薦標準。CSS3現在已被大部分現代瀏覽器支援，而下一般的CSS4仍在開發中。

JavaScript-負責行為、互動


JavaScript，通常縮寫為JS，是一種進階直譯語言的程式語言。JavaScript是一門基於原型、函式先行的語言，是一門多範例式的語言，他支援物件導向編程，指令是程式設計，以及函式語言程式設計。
